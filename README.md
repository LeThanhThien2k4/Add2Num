# Big Number Sum - Spring Boot

## Giới thiệu
Ứng dụng web cho phép cộng 2 số lớn (string) bằng thuật toán giống học sinh tiểu học.

## Công nghệ
- Java
- Spring Boot
- Thymeleaf
- Bootstrap

## Chức năng
- Nhập 2 số bất kỳ (dạng chuỗi)
- Cộng số lớn không giới hạn độ dài
- Logging từng bước phép cộng
- Validate input chỉ cho phép số

## Cách chạy project

### 1. Clone project
git clone https://github.com/youraccount/projectname

### 2. Mở bằng IntelliJ

### 3. Run
Chạy file:
Application.java

### 4. Truy cập
http://localhost:8080

## Test case

| Input        | Output |
|-------------|--------|
| 1234 + 897  | 2131   |
| 999 + 1     | 1000   |

## Logging
Ứng dụng ghi log từng bước cộng số trong console.
